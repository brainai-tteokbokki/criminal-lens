Documentation

authKey is not a required parameter, but if you do not provide it to the server or it is incorrect, you will not be able to receive face data.

/api/list

Retrieve registered account information.

Request - Method: POST
Key Description
authKey Authentication Key. (string)
Response
Key Description
error Did an error occur? (bool)
detail Status detail. (string)
/api/search

API to find out what object is in the image

Request - Method: POST
Key Description
authKey Authentication Key. (string)
crimi_face Find the criminal's face. (base64. string)
Response
Key Description
similar_info Registered criminal suspected of resemblance. (object)
suspect_info Registered offender suspected of being a suspect. (object)
error Did an error occur? (bool)
detail Status detail (str)
/api/regi

Register criminals to use when searching.

Request - Method: POST
Key Description
authKey Authentication Key. (string)
crimi_name Name of the offender to be registered. (string)
crimi_desc The crime and description of the offender. (string)
crimi_image The face of the criminal to be registered (base64. string)
is_agree Agree to the processing of personal information. (bool)
regi_time Registration time. (%Y%m%d_%H%M%S. string)
Response
Key Description
error Did an error occur? (bool)
detail Status detail (str)
